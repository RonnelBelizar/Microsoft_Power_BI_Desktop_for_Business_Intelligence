# Microsoft Power BI Learning Journal

This repository is my personal learning journal documenting my journey through the Microsoft Power BI course. It captures my reflections, notes, exercises, and projects as I build practical skills in data analysis and business intelligence.

## Repository Structure

* **notes/**: Contains my personal journal entries, learning notes, and reflections for each module of the Microsoft Power BI course. Each entry details concepts learned, challenges faced, insights gained, and how I applied them in practice.
* **exercises/**: Contains all hands-on exercises, sample datasets, and Power BI dashboard files that I created while following the course. This includes data cleaning, transformation, modelling, DAX calculations, and visualization projects.

## Key Skills Documented

* **Data Cleaning & ETL:** How I prepared and transformed raw datasets into analysis-ready formats using Power Query.
* **Data Modelling:** Designing relationships between tables, creating star schemas, and structuring data for optimal dashboard performance.
* **DAX & Calculations:** Writing essential DAX formulas like CALCULATE, SUMX, FILTER, and time-intelligence functions.
* **Dashboard Design & Visualization:** Creating interactive, insightful dashboards with slicers, filters, and user-friendly visuals.
* **Business Analysis Insights:** Translating data into actionable insights, understanding KPIs, and exploring trends.

## How to Use This Repository

* Browse the `notes/` folder to see detailed reflections and learning progress for each course module.
* Explore `exercises/` to view all dashboards, datasets, and practice exercises I completed.
* This repository serves as a record of my learning process and showcases my capability to apply Power BI skills to real-world scenarios.
